# 🧭 Rol: Scrummaster (SM)

## Verantwoordelijkheden
De Scrummaster bewaakt het scrumproces en ondersteunt het team.

- Sprint planning faciliteren (🌑 Nieuwe Maan)
- Dagelijkse standup leiden
- Obstakels voor het team wegnemen
- Retrospectief organiseren (🌗 Laatste Kwartier)
- Sprint afsluiten (🌘 Afnemende Sikkel)

## Maanfase activiteiten
| Fase | Activiteit |
|------|-----------|
| 🌑 Nieuwe Maan | Sprint planning leiden, taakverdeling maken |
| 🌓 Eerste Kwartier | Code reviews begeleiden, standups houden |
| 🌕 Volle Maan | Demo uitvoeren |
| 🌗 Laatste Kwartier | Retrospectief faciliteren, verbeterpunten noteren |
| 🌘 Afnemende Sikkel | Sprint officieel afsluiten |
