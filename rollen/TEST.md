# 🧪 Rol: Tester (TEST)

## Verantwoordelijkheden
De Tester waarborgt de kwaliteit van het opgeleverde werk.

- Testplannen opstellen
- Tussentijds testen tijdens de ontwikkelfase
- Test rapport schrijven na de sprint review
- Hertest uitvoeren na bugfixes

## Maanfase activiteiten
| Fase | Activiteit |
|------|-----------|
| 🌓 Eerste Kwartier | Tussentijds testen uitvoeren |
| 🌕 Volle Maan | Test rapport schrijven |
| 🌖 Afnemende Gibbous | Hertest na bugfixes |
