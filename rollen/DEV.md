# 💻 Rol: Developer (DEV)

## Verantwoordelijkheden
De Developer bouwt de features en lost technische problemen op.

- Ontwikkelomgeving klaarzetten
- Features bouwen op basis van user stories
- Code reviews uitvoeren
- Bugs oplossen na feedback

## Maanfase activiteiten
| Fase | Activiteit |
|------|-----------|
| 🌒 Wassende Sikkel | Dev-omgeving klaarzetten, eerste taken starten |
| 🌓 Eerste Kwartier | Features ontwikkelen, code reviews |
| 🌕 Volle Maan | Demo voorbereiden samen met PO |
| 🌖 Afnemende Gibbous | Bugfixes op basis van feedback |
