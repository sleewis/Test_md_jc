# 🌕 Scrumbord — Maanfasen Sprint

Een scrumboard waarbij de **maancyclus (28 dagen)** de sprint definieert.
Elke maanfase heeft een eigen focus en verantwoordelijkheden.

---

## 🌑 Nieuwe Maan — Sprint Planning
**Duur:** dag 1–3
**Focus:** Backlog refinement, sprint planning, doelen stellen

| Taak | Rol | Status |
|------|-----|--------|
| Sprint doelen bepalen | PO + SM | ⬜ |
| User stories opstellen | PO | ⬜ |
| Taakverdeling maken | SM | ⬜ |

---

## 🌒 Wassende Sikkel — Opstarten
**Duur:** dag 4–7
**Focus:** Omgeving klaarzetten, eerste taken oppakken

| Taak | Rol | Status |
|------|-----|--------|
| Dev-omgeving klaarzetten | DEV | ⬜ |
| Eerste taken in uitvoering | DEV | ⬜ |
| Documentatie structuur opzetten | DOC | ⬜ |

---

## 🌓 Eerste Kwartier — Ontwikkeling
**Duur:** dag 8–14
**Focus:** Hoofd-ontwikkelperiode, dagelijkse standups

| Taak | Rol | Status |
|------|-----|--------|
| Features ontwikkelen | DEV | ⬜ |
| Code reviews uitvoeren | DEV + SM | ⬜ |
| Tussentijds testen | TEST | ⬜ |
| Voortgang documenteren | DOC | ⬜ |

|------|-----|--------|
