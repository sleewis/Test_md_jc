# 🌕 Test_md_jc — Maanfasen Scrumproject

Een scrumproject waarbij de **maancyclus van 28 dagen** de sprint vormt.

## Structuur

```
Test_md_jc/
├── README.md                    # Dit bestand
├── SCRUMBORD.md                 # Het actieve scrumbord
├── Test_md_jc_instructies.md    # Projectinstructies
├── rollen/                      # Rolomschrijvingen
│   ├── SM.md                    # Scrummaster
│   ├── DEV.md                   # Developer
│   ├── TEST.md                  # Tester
│   ├── PO.md                    # Product Owner
│   └── DOC.md                   # Documenteerder
└── sprints/                     # Sprint verslagen (per cyclus)
```

## Maancyclus als Sprint

| Maanfase | Dagen | Scrum activiteit |
|----------|-------|-----------------|
| 🌑 Nieuwe Maan | 1–3 | Sprint Planning |
| 🌒 Wassende Sikkel | 4–7 | Opstarten |
| 🌓 Eerste Kwartier | 8–14 | Ontwikkeling |
| 🌕 Volle Maan | 15–17 | Review & Demo |
| 🌖 Afnemende Gibbous | 18–21 | Verfijnen |
| 🌗 Laatste Kwartier | 22–25 | Retrospectief |
| 🌘 Afnemende Sikkel | 26–28 | Afsluiten |

## Rollen
- **SM** — Scrummaster: bewaakt het proces
- **DEV** — Developer: bouwt de features
- **TEST** — Tester: waarborgt de kwaliteit
- **PO** — Product Owner: beheert de backlog
- **DOC** — Documenteerder: legt alles vast
