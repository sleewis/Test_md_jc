## Doel van het project
We bouwen samen een product bestaande uit **inzichtkaarten en dobbelstenen gebaseerd op heilige geometrie**.

De ontwikkeling wordt begeleid via een **scrumbord op basis van maanfasen**: de maancyclus van 28 dagen vormt één sprint. Elke fase van de maan heeft een eigen rol binnen het scrumproces.

## Scrumstructuur (maanfasen)
| Maanfase | Scrum activiteit |
|----------|-----------------|
| 🌑 Nieuwe Maan | Sprint Planning |
| 🌒 Wassende Sikkel | Opstarten |
| 🌓 Eerste Kwartier | Ontwikkeling |
| 🌕 Volle Maan | Review & Demo |
| 🌖 Afnemende Gibbous | Verfijnen |
| 🌗 Laatste Kwartier | Retrospectief |
| 🌘 Afnemende Sikkel | Afsluiten |
